Search Engine Optimization (SEO)

Rank - Improvement Tips

  • Have relevant conten on your page
  • Use keywords in your content
  • Mobile-friendly - Avoid popups
  • Content accuracy
  • How fast your page loads
  • Page engagement - How long do users stay
  • Page accessibility
  • Custom 404 Page
  • Proper meta tags
  • Use HTTPS
  • Use proper headings - one h1, multiple h2 - h6

Page Status Codes

  1. 1xx - Still working
  2. 2xx - All good
  3. 3xx - Redirects
  4. 4xx - You made a mistake
  5. 5xx - The server had an issue
